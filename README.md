## Dev Features

- Next.js

## Development

1. Clone the repo:

```bash
git clone git@github.com:gregrickaby/kindergarten-flashcards.git
```

2. Install dependencies:

```bash
yarn install
```

3. Start the development server:

```bash
yarn dev
```
